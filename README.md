# remove-files

_Elimina todos los archivos, que cumplan con la extension ingresada. Dentro del directorio ingresado, incluyendo todas las subcarpetas de este._

### Pre-requisitos 📋

_Herramientas necesarias para correr el script_

```
Python 3.9
```
